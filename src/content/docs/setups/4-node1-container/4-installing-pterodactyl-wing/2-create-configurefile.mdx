## Create Subdomain
I have moved this section to the <a target="_blank" href="../../../subdomains/subdomain-node1">`Subdomains`</a>.<br/>
You need this step to continue with the setup as well as to log in at all.

Then we start with the creation of our Configure-File.
* Open `webui.yourdomain.com`
* Go to the Settings (Click on the gear-symbol)

## Create new Location
* Switch to `Managment > Location` (Left Sidebar)
* Hit blue `Create New`-Button
* Enter the following values
* Hit green `Create`-Button

### Locationsetting
<Tabs>
    <TabItem label="Create Location">
        Short Code      -   Individual personal code. I use the Location from Hetzner like `FSN1-DC11`<br/>
        Description     -   Empty<br/>
    </TabItem>
</Tabs>

## Create new Node
* Switch to `Managment > Nodes` (Left Sidebar)
* Hit blue `Create New`-Button
* Enter the following values
* Hit green `Create Node`-Button

### Nodesetting
<Tabs>
    <TabItem label="Basic Details">
        Name                    -   Name for this Server, example: `private node`<br/>

        Description             -   Empty<br/>

        Location                -   Your location name, which you have just created<br/>

        Node Visibility         -   `Private` (Option to decide whether it is public or private)<br/>

        FQDN                    -   Domain or IP (I use the subdomain `node1.yourdomain.com`)<br/>

        Communicate Over SSL    -   `Use SSL Connection`<br/>

        Behind Proxy            -   `Behind Proxy`<br/>
    </TabItem>
    <TabItem label="Configuration">
        Daemon Server File Directory    -   change nothing<br/>

        Total Memory                    -   `1024MiB - 8192MiB` (1GB - 8GB | I use `7168MiB`)<br/>

        Memory Over-Allocation          -   `0%`<br/>

        Total Disk Space                -   `1024MiB - 25600MiB` (1GB - 25GB | I use `20480MiB`)<br/>

        Disk Over-Allocation            -   `0%`<br/>

        Daemon Port                     -   `443`<br/>

        Daemon SFTP Port                -   `2022`<br/>
    </TabItem>
</Tabs>

## Set config-filepath
* Switch to `Managment > Nodes` (Left Sidebar)
* Click on your first node
* Switch to `configuration`-Tab
<br/>
* Open in a new tap `proxmox.yourdomain.com`
* Go to Node-CT-201-Console
* Create a new config file `nano /etc/pterodactyl/config.yml`
* Paste the Text from the `Configure-Tab` into this file
* Save (CTRL+o) and Exit (CTRL+x)
<br/>
* Go back to the `dockge.yourdomain.com`
* Go to your `pterodactyl-wing-node1-container`
* Click `Stop`-Button
* Click `Start`-Button

<Aside type="danger">
    Don't use the `restart`-button
</Aside>
