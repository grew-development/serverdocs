## Connecting to Dockge-Web-UI
* Open `dockge.yourdomain.com`

## Create DockerCompose-file
* Click `+ Compose`-Button
* Enter the following values
* Paste the <a target="_blank" href="../../../configs/docker-composefiles/compose-pterodactyl">docker compose file</a> in the right window
* Scroll down to '.env-Area'
* Copy the variable name into it and give it a value
* Hit save-button

#### Composesetting
<Tabs>
    <TabItem label="General">
        Stack-Name      -   Name of your compose (example: pterodactyl-panel)<br/>

    </TabItem>
    <TabItem label="Containers">
        Change nothing<br/>
    </TabItem>
